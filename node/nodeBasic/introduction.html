          # 二、Buffer

          # 概念

          Buffer中文译为缓冲区，是一个类似于Array的对象，用于表示固定长度的字节序列。换句话说就是一段固定长度的内存空间，用于处理二进制数据

          # 特点

          1.Buffer大小固定，且无法调整

          2.Buffer性能较好，可以直接对计算机内存进行操作

          3.每个元素的大小为1字节（byte）
